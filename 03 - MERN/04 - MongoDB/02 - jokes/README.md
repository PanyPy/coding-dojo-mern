# FAKE API

## Install dependencies
`npm install`

## Start server
`npm run dev` or `nodemon server.js`

## API
`/api/users/new` => Returns a new random User 
`/api/companies/new` => Returns a new random Company 
`/api/user/company` =>  returns a new random {User, Company}
